盒模型

這是一個盒模型範例

解釋：

內容區域：寬度為 200px，高度為 100px。

內邊距：每邊 20px，將內容與邊框之間隔開。

邊框：5px 寬的黑色實線，包圍在內邊距的外側。

外邊距：每邊 10px，將該元素與周圍其他元素隔開。

總寬度 = 200px（內容） + 40px（內邊距） + 10px（邊框） + 20px（外邊距） = 270px

總高度 = 100px（內容） + 40px（內邊距） + 10px（邊框） + 20px（外邊距） = 170px

盒模型 範圍

Content-box
Border-box

解釋：

第一個方框使用 box-sizing: content-box，因此它的總寬度和高度會超過 200px x 100px。

第二個方框使用 box-sizing: border-box，因此它的總寬度和高度會精確地符合 200px x 100px 的設定。
